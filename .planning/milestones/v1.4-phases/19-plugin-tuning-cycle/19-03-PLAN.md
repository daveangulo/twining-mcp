  <action>
Create `test/eval/regression-baseline.ts` -- a Node script (run via tsx) that:

1. **Reads eval results** from `test/eval/results/latest.json` (synthetic), `test/eval/results/holdout-latest.json`, and `test/eval/results/transcript-latest.json`
2. **Reads thresholds** from scorer-types.ts (import SCORER_THRESHOLDS)
3. **Reads plugin version** from `plugin/.claude-plugin/plugin.json` (version field)
4. **Measures token budget** by reading all plugin artifact files and summing byte counts (same logic as measure-plugin-tokens.sh but in TypeScript)
5. **Reads scenario YAML files** to include tags and holdout flags per scenario
6. **Assembles the RegressionBaseline object:**

```typescript
import fs from "node:fs";
import path from "node:path";
import { SCORER_THRESHOLDS } from "./scorer-types.js";

interface RegressionBaseline {
  version: "1.0";
  timestamp: string;
  plugin_version: string;
  thresholds: Record<string, number>;
  token_budget: {
    pre_tuning_bytes: number;
    post_tuning_bytes: number;
    growth_percent: number;
    cap_bytes: number;
  };
  aggregate: {
    synthetic_pass_rate: number;
    synthetic_overall_score: number;
    transcript_pass_rate: number;
    transcript_overall_score: number;
    holdout_pass_rate: number;
    holdout_overall_score: number;
  };
  scenarios: Array<{
    name: string;
    category: string;
    tags: string[];
    holdout: boolean;
    scores: Record<string, {
      score: number;
      passed: boolean;
      threshold: number;
      checks_passed: number;
      checks_total: number;
    }>;
  }>;
}
```

7. **Writes** the baseline to `test/eval/results/regression-baseline.json`
8. **Prints** a summary to stdout: pass rates, score averages, token budget

The script should be runnable via: `npx tsx test/eval/regression-baseline.ts`
