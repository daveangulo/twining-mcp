   **Types:**
   - `JudgeResult`: `{ level: string; reasoning: string }`
   - `ConsensusResult`: `{ score: number; level: string; agreement: string; trials: JudgeResult[] }`

   **Constants:**
   - `LEVEL_SCORES`: `Record<string, number>` mapping `good: 1.0, acceptable: 0.5, bad: 0.0`
   - `JUDGE_TOOL`: Tool schema with `name: "judge_result"`, `input_schema` requiring `level` (enum: good/acceptable/bad) and `reasoning` (string)

   **Functions:**
   - `levelToScore(level: string): number` -- maps level string to 0-1 score via LEVEL_SCORES, defaults to 0.0 for unknown
   - `median(values: number[]): number` -- sorts, returns middle value (or average of middle two for even arrays)
   - `createJudgeClient(): Anthropic | null` -- returns null if `ANTHROPIC_API_KEY` is not set, otherwise `new Anthropic()` (SDK auto-reads key)
   - `callJudge(client: Anthropic, prompt: string): Promise<JudgeResult>` -- calls `client.messages.create` with model from `TWINING_EVAL_MODEL ?? "claude-haiku-4-5-20250929"`, `max_tokens: 512`, `temperature: 0`, `tool_choice: { type: "tool", name: "judge_result" }`, `tools: [JUDGE_TOOL]`. Extracts tool_use block defensively (find where `type === "tool_use"`). Throws if no tool_use block found.
   - `consensusScore(judgeFn: () => Promise<JudgeResult>, k?: number): Promise<ConsensusResult>` -- reads `k` from param or `TWINING_EVAL_TRIALS` env var (default 3). Fires k trials in parallel via `Promise.all`. Counts level occurrences, finds majority (>= ceil(k/2)). If majority exists, use `levelToScore(majorityLevel)`. Otherwise use `median(scores)`. Returns `ConsensusResult` with agreement like "2/3" or "3/3".
   - `buildRubricPrompt(criterion: QualityCriterion, context: string): string` -- constructs a prompt with the criterion name, rubric levels formatted as `- **{level}**: {description}\n  Example: {example}`, and the tool call context to evaluate. Import `QualityCriterion` from `./types.js`.

   **Error handling:** `callJudge` should let SDK errors propagate (callers handle). `createJudgeClient` is the safety gate.
